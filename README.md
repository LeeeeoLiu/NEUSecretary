## 立题背景
- 微信东大小秘书使用比查看网页教务处方便不少
- Siri 可语音安排日程查询等
- 微软小娜
- ...

## 我们要实现的目标:
做一个东大小秘书的桌面版,它应该包含的基本功能有:
- 图书馆
- 查询课表
- 上自习
- 蹭课
- 绑定学号
- 考场
- 查成绩
- 查看校历
- 查看校车
- ...


它应该包含的扩展功能有:
- 语音识别,比如语音查询成绩,制定计划等
- 支持与其他人聊天
- 桌面小精灵
- 课程活动提示
- 实时更新天气消息
- ...
